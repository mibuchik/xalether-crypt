Инструмент каскадного шифрования файлов и папок с современным GUI на PyQt5.

---

## Возможности

| Функция | Описание |
|---|---|
| 🔐 Каскадное шифрование | AES-256-GCM → ChaCha20-Poly1305 |
| 📁 Поддержка папок | Автоматическая упаковка в ZIP → шифрование |
| 🔒 Личный режим | Файл привязан к HWID текущего ПК |
| 📤 Режим передачи | Без ограничений — любой с паролем |
| 🎫 Режим разрешений | Одноразовые коды (24 ч, 1 использование) |
| ⬇ Drag-and-Drop | Перетаскивание файлов и папок прямо в окно |
| 🔑 Генератор паролей | Криптографически стойкие пароли |
| 📋 История операций | Сохраняется в `~/.xalether_history.json` |
| ⚙ Настройки | Режим по умолчанию, авто-удаление оригиналов |
| 🧵 Многопоточность | Шифрование в фоне через QThread |

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Mibuchik/xalether-crypt.git
cd xalether-crypt

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить
python src/main.py
```

## Алгоритм шифрования

```
Данные
  ↓ zlib compress (level 9)
  ↓ AES-256-GCM  (iv: 12 байт, tag: 16 байт, AAD: "XALETHER_CRYPT_V2")
  ↓ ChaCha20-Poly1305 (iv: 12 байт, AAD: "XALETHER_CRYPT_V2")
  ↓ + случайный шум (16–79 байт в начале)
```

Деривация ключей: **PBKDF2-SHA256**, 500 000 итераций, 64-байтный мастер-ключ.

---

## Формат файла `.xalether`

| Поле | Размер | Описание |
|---|---|---|
| meta_len | 4 байта | Длина JSON-метаданных |
| metadata | meta_len | JSON (версия, режим, HWID/код) |
| salt | 16 байт | Соль для PBKDF2 |
| ciphertext | остальное | Зашифрованные данные |

## ⚠ Предупреждение о безопасности

- **Мастер-пароль не восстанавливается**. Потеря пароля = потеря данных.
- Файлы в режиме «Личный» открываются только на том компьютере, где были зашифрованы.
- Коды разрешений — одноразовые. После использования автоматически удаляются.
- Не передавайте мастер-пароль вместе с зашифрованными файлами.
